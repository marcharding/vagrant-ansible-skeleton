# Vagrant-Ansible Skeleton

## Available roles

- apache-fpm
- apache-mod
- app
- composer
- dotfiles
- init
- java
- magerun
- modman
- mysql
- nginx
- oxrun
- php5-fpm
- php5-mod
- ruby
- solr
- install-oxid-esales

## Usage

Adjust ip adresses and hostname, choose your roles and run `vagrant up`.
